stageset-controller {{ .Chart.AppVersion }} is installed in namespace {{ .Release.Namespace }}.

The controller reconciles StageSet resources cluster-wide. Create one with:

  kubectl apply -f - <<'EOF'
  apiVersion: stages.metio.wtf/v1
  kind: StageSet
  metadata:
    name: example
    namespace: {{ .Release.Namespace }}
  spec:
    serviceAccountName: my-deployer
    stages: []
  EOF
